Accessible Data Asset Components

This page demonstrates the accessible data asset card components with keyboard navigation and screen reader support.

Keyboard Navigation Instructions

These cards are fully accessible with the following keyboard shortcuts:

  • Tab - Move focus to the next card or interactive element
  • Enter or Space - Activate a card to edit it
  • Escape - Cancel editing mode
  • Ctrl + Enter - Save changes in edit mode

Screen reader announcements will guide you through the interaction process.
